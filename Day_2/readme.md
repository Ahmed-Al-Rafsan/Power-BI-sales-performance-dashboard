# Power BI Sales Performance Dashboard



This project focuses on building a clean, business-ready sales performance dashboard using Power BI.



**Key Features**

\- KPI cards for Total Sales, Total Profit, and Total Orders

\- Sales trend analysis over time (monthly)

\- Sales breakdown by Category

\- Profit analysis by Region

\- Interactive slicers (Category, Region, Date)



**Skills Demonstrated**

\- Data modeling and preparation

\- Business-focused KPI design

\- Time-series analysis

\- Interactive dashboard design

\- Clean and professional layout



**Dataset**

\- Superstore Sales Dataset



**Purpose**

This project is part of my Power BI Bootcamp for Data Analytics, aimed at developing job-ready dashboarding and analytical thinking skills.
